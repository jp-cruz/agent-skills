🚀 WHAT IT DOES
═══════════════════════════════════════════════════════════════════════════

Provides production-ready Docker Compose setup for Thoth with:

  ✅ Cross-Platform Support
     • macOS (Intel & Apple Silicon) → uses host.docker.internal
     • Windows (WSL 2) → uses host.docker.internal
     • Linux → uses local IP or remote Ollama IP

  ✅ Zero Configuration (sensible defaults)
     • Automatic setup scripts (setup.sh / setup.bat)
     • Validates prerequisites (Docker, docker-compose, Ollama)
     • Creates .env automatically
     • Creates data directories

  ✅ Full Parameterization
     • THOTH_PORT — Server port
     • THOTH_DATA_DIR — Data persistence
     • THOTH_WORKSPACE_DIR — Workspace files
     • OLLAMA_BASE_URL — LLM endpoint
     • RESTART_POLICY — Container restart

  ✅ Development Utilities (installed as root, accessible to thoth user)
     • nano — Text editor for config files
     • vim-tiny — Vi implementation for power users
     • jq — JSON processor for API debugging (CRITICAL)
     • less — File pager for logs
     • file — File type identification
     • tree — Directory visualization
     • unzip — Archive extraction
     + curl, git, ffmpeg, gcc (pre-existing)

  ✅ Production Safety
     • Non-root user (thoth, UID 1000)
     • Persistent volumes (bind mounts)
     • Auto-restart on failure
     • Ollama integration tested
     • Minimal footprint (~300 MB)

  ✅ Comprehensive Documentation
     • Platform-specific setup instructions
     • Troubleshooting guides
     • Utility analysis and rationale
     • Developer guide (CLAUDE.md)
     • Publishing instructions
     • Multi-machine deployment examples


🎯 QUICK START
═══════════════════════════════════════════════════════════════════════════

macOS/Linux:
  $ git clone https://github.com/jp-cruz/agent-skills
  $ cd agent-skills/thoth-docker-setup
  $ ./setup.sh
  $ vim .env          # customize if needed
  $ docker-compose up -d
  $ open http://localhost:8080

Windows (PowerShell):
  > git clone https://github.com/jp-cruz/agent-skills
  > cd agent-skills\thoth-docker-setup
  > setup.bat
  > notepad .env
  > docker-compose up -d
  > start http://localhost:8080


🛠️ COMMON COMMANDS
═══════════════════════════════════════════════════════════════════════════

docker-compose up -d                    # Start
docker-compose stop                     # Stop
docker-compose logs -f                  # View logs
docker-compose ps                       # Check status
docker-compose exec thoth bash          # Open shell

# Edit config inside container
docker-compose exec thoth nano /home/thoth/.thoth/config.yaml

# Debug Ollama connectivity with jq
docker-compose exec thoth curl -s http://host.docker.internal:11434/api/tags | jq '.'

# Explore workspace with tree
docker-compose exec thoth tree -L 2 /home/thoth/.thoth
